## Projetos

A seguir, os principais projetos organizados por área.

### 1. SQL

Projetos focados em criação de KPIs, consultas avançadas, joins, window functions e análise exploratória usando SQL puro.

* **Análise de Vendas – Superstore (SQL)**
  KPIs, CTEs, joins, agregações, análise de performance.

* **Window Functions – Ranking e Séries Temporais**
  Uso de RANK, LAG, LEAD e partições para análises analíticas.

Acesse os projetos na pasta `/sql`.

---

### 2. Power BI

Dashboards profissionais com modelagem dimensional, DAX intermediário/avançado e storytelling.

* **Dashboard Executivo – Superstore**
  Modelagem star schema, KPIs de vendas, comparação temporal e visual analyics entregues em PBIX.

Acesse os dashboards na pasta `/power-bi`.

---

### 3. Python

Projetos em Jupyter Notebook com ETL, limpeza, transformação, EDA e visualizações.

* **ETL – Superstore**
  Pipeline de transformação e enriquecimento dos dados.

* **EDA – E-commerce Brasileiro (Olist)**
  Insights, gráficos, comportamento de clientes e análise operacional.

Acesse em `/python`.

---

### 4. Projeto Integrado (SQL + Python + Power BI)

Um case completo unindo todas as etapas:

* Extração e métricas iniciais em SQL
* Limpeza e prepararção em Python
* Visualização final no Power BI

Localizado em `/integrated-project`.
